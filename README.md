# Gestão de Motoboys

Este projeto consiste em uma tela de gestão de motoboys desenvolvida com HTML, CSS e JavaScript. A tela permite visualizar a lista de motoboys cadastrados, selecionar um motoboy para visualizar suas informações pessoais e inclui botões de editar e excluir para cada motoboy.

## Desafio 1: JavaScript

Para resolver o Desafio 1, foi criado um código JavaScript que recebe um array de usuários contendo nome, idade e email e uma string de busca. O código utiliza uma promise para percorrer todos os usuários e retorna o primeiro que tenha o valor de busca em alguma das propriedades. Se nenhum usuário for encontrado, o código trata o erro e o apresenta pelo catch.

## Desafio 2: HTML + CSS

Para resolver o Desafio 2, foi desenvolvida uma interface de usuário (UI) utilizando HTML e CSS. A tela exibe a lista de motoboys cadastrados, permitindo a visualização de suas informações pessoais ao selecionar um motoboy. Além disso, foram adicionados botões de editar e excluir para cada motoboy, mesmo que as funcionalidades de edição e exclusão não tenham sido implementadas.

**Instruções de Uso:**

1. Faça o clone deste repositório.
2. Abra o arquivo index.html em seu navegador para visualizar a tela de gestão de motoboys.

**Observação:** As funcionalidades de edição e exclusão não estão implementadas nesta versão, apenas a interface estática para visualização dos motoboys cadastrados.

Este projeto foi desenvolvido como parte de um desafio e pode ser expandido com funcionalidades adicionais e melhorias na UI/UX.
